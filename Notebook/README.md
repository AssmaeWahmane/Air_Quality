Introduction

This project was developed as part of the Master's Degree in Econometrics and Statistics at the Institut de Science Financière et d'Assurances (ISFA), Claude Bernard University Lyon 1.

The objective of the project was to develop practical skills in web scraping, data manipulation, version control, and software engineering best practices using Python. The project emphasizes the use of key Python libraries such as Pandas and NumPy, Git for version control, and a modular code structure following the PEP 8 coding standards.

The dataset was collected through the Open-Meteo API and includes information on air quality (various pollutants) and climate conditions across different regions of France.
